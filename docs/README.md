# Veyra Docs

Veyra is a standalone Minecraft launcher plus a client-side Fabric utility mod.

## Parts

- [`launcher.md`](launcher.md) — Tauri/Svelte/Rust launcher
- [`mod.md`](mod.md) — Fabric client mod features and keybinds
- [`build.md`](build.md) — build/run notes

- [`roadmap.md`](roadmap.md) — safer feature ideas
